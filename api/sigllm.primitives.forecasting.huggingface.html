sigllm.primitives.forecasting.huggingface module¶

class sigllm.primitives.forecasting.huggingface.HF(name='mistralai/Mistral-7B-Instruct-v0.2', sep=',', steps=1, temp=1, top_p=1, raw=False, samples=1, padding=0)[source]¶

Bases: object

Prompt Pretrained models on HuggingFace to forecast a time series.

Parameters

  • name (str) – Model name. Default to ‘mistralai/Mistral-7B-Instruct-v0.2’.

  • sep (str) – String to separate each element in values. Default to ‘,’.

  • steps (int) – Number of steps ahead to forecast. Default 1.

  • temp (float) – The value used to modulate the next token probabilities. Default to 1.

  • top_p (float) – If set to float < 1, only the smallest set of most probable tokens with probabilities that add up to top_p or higher are kept for generation. Default to 1.

  • raw (bool) – Whether to return the raw output or not. Defaults to False.

  • samples (int) – Number of forecasts to generate for each input message. Default to 1.

  • padding (int) – Additional padding token to forecast to reduce short horizon predictions. Default to 0.
