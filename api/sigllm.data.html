sigllm.data module¶

Data Management module.

This module contains functions that allow downloading demo data from Amazon S3, as well as load and work with other data stored locally.

sigllm.data.download_normal(name, data_path='/home/runner/work/sigllm/sigllm/sigllm/data')[source]¶

Load the CSV with the given name from S3.

If the CSV has never been loaded before, it will be downloaded from the [sintel-sigllm bucket](https://sintel-sigllm.s3.amazonaws.com) or the S3 bucket specified following the s3://{bucket}/path/to/the.csv format, and then cached inside the data folder, within the sigllm package directory, and then returned.

Otherwise, if it has been downloaded and cached before, it will be directly loaded from the sigllm/data folder without contacting S3.

Parameters

  • name (str) – Name of the CSV to load.

  • data_path (str) – Path to store data.

Returns

A pandas.DataFrame is returned containing all the data.

Return type

pandas.DataFrame

Raises

  • FileNotFoundError – If the normal file doesn’t exist locally and can’t

  • be downloaded from S3. –

Load normal data from file or download if needed.

Parameters

  • name (str) – Name or path of the normal data.

  • timestamp_column (str or int) – Column index or name for timestamp.

  • value_column (str or int) – Column index or name for values.

  • start (int or timestamp) – Optional. If specified, this will be start of the sub-sequence.

  • end (int or timestamp) – Optional. If specified, this will be end of the sub-sequence.

Returns

Loaded subsequence with timestamp and value columns.

pandas.DataFrame
