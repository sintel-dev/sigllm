format_as_integer(X, separator=',', trunc=None, digits_per_timestamp=3, target_column=None, **kwargs)[source]¶

Parse interleaved value strings back to integer arrays for the target dimension.

Parameters

  • X (list[str]) – list of strings, each string is a concatenation of num_dims values separated by separator.

  • separator (str) – separator between values

  • trunc (int) – Number of values to extract from each sample. If None, all values are extracted.

  • digits_per_timestamp (int) – Number of digits to extract from each timestamp.

  • target_column (int) – Which dimension to extract (default 0). Can also be set via config.

Returns

Array that holds int values for the target column for each sample in each window.

Return type

np.ndarray
